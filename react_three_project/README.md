# 🚀 HCI Project: Interactive 3D Festive Gift Box

An interactive 3D web application built for my **HCI (Human-Computer Interaction)** course. This project focuses on intuitive 3D environmental interaction and visual feedback.

## ✨ Technical Highlights
* **Dynamic Texture Mapping**: Custom logic to upload local images onto 3D cube faces in real-time.
* **Layered Particle Systems**: A high-density "sparkle" effect using white "hero stars" and gold dust to simulate realistic fireworks.
* **Mouse-Responsive Parallax**: Smooth 3D rotation that follows the user's cursor for an immersive feel.
* **State-Driven Animation**: Sliding box lid animations triggered by user interaction.

## 🛠️ Tech Stack
* **Framework**: React.js (Vite)
* **3D Library**: Three.js / React Three Fiber
* **Utilities**: @react-three/drei (for Sparkles and Text)
