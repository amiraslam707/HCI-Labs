# HCI Button Lab – Lab 05 (a)

## Goal
Refactored the inline-style and inline-script button example from Lab 03/04 to follow **separation of concerns**:

- **Structure** → `index.html`
- **Presentation/Style** → `style.css`
- **Behavior** → `script.js`

## HCI Principles demonstrated
- Affordance: blue border on button
- Mapping: green status area
- Feedback: yellow flash on click
- Constraint: button disables after 3 clicks
- Signifier: purple color + opacity when disabled

Open `index.html` in a browser to test.
